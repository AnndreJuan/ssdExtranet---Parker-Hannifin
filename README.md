## Descrição do Projeto

Este projeto tem como objetivo cadastrar o estoque dos distribuidores no canal da Parker Hannifin (https://www.extranetparker.com.br) de forma automática e periódica, garantindo que os dados estejam sempre atualizados e disponíveis para outros distribuidores.

---

## Configuração do Ambiente

### Criação e Ativação do Ambiente Virtual

python -m venv

### Requirements

pip install -r requirements

## Adendo

Este projeto foi desenvolvido utilizando a API do Maxiprod (ERP), a qual requer uma chave de acesso para sua utilização. Se desejar utilizar outra API, será necessário adaptar o projeto para a sua.
